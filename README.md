## Cómo instalar en Termux:

`$ pkg up -y`

`$ pkg install openssl-tool`

`$ pkg install curl`

`$ pkg install git`

`$ git clone https://github.com/ze4n-github/INSTA-BOT`

`$ cd INSTA-BOT`

`$ termux-wake-lock`

`$ python INSTA-BOT.py`



## Cómo instalar en Kali o Parrot:
`$ git clone https://github.com/ze4n-github/INSTA-BOT`

`$ cd INSTA-BOT`

`$ python INSTA-BOT.py`
